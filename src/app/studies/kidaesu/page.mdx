# Basics to Grad Math (KiDaeSu) 강의 링크 색인

이 페이지는 조건희 교수님의 **Basics to Grad Math (KiDaeSu)** 시리즈를 공부하기 위한 강의 링크 색인이다.

KiDaeSu는 집합론에서 시작해 해석학, 위상수학, 선형대수학, 추상대수학, 측도론, 다변수해석학으로 이어지는 수학 전공 학습 경로를 제공한다. 이 페이지에서는 각 강의의 제목과 영상 링크를 한곳에 모아두고, 이후 별도의 공부 기록·개념 정리·증명 노트와 연결하기 위한 기준 목차로 사용한다.

## 공식 출처

- 공식 페이지: [Basics to Grad Math](https://sites.google.com/view/enjoyingmath/home/basics-to-grad-math?authuser=0)
- 강의자: 조건희 교수님
- 시리즈명: Basics to Grad Math, KiDaeSu
- 개인 학습 자료: KDS notes 1–120

## 사용 방식

이 페이지는 강의 내용을 정리하는 곳이 아니라, 원자료에 접근하기 위한 색인이다.
각 강의에 대한 실제 공부 기록은 별도의 글로 작성하고, 이 페이지에서는 해당 글을 이후 연결한다.

예상 연결 구조는 다음과 같다.

| 항목 | 역할 |
| ---- | ---- |
| KDS 강의 링크 색인 | 전체 영상 목록과 링크 관리 |
| KDS 강의별 공부 기록 | 각 강의의 핵심 흐름 요약 |
| 개념 노트 | 정의, 예시, 반례, 연결 개념 정리 |
| Proof Lab | 주요 정리와 증명의 재작성 |
| Study Log | 실제 공부 과정, 막힌 지점, 복습 기록 |

## 표기 메모

- `KDS-000`은 본강의가 아니라 시리즈 후기 영상이다.
- 본강의는 `KDS-001`부터 `KDS-120`까지로 둔다.
- 일부 강의 제목의 괄호 표기 `(j)`, `(k)`, `(l)` 등은 원자료 제목을 따른다.
- 이후 각 강의별 페이지가 만들어지면 이 표에 `정리글 링크`, `KDS 노트 페이지`, `개념 태그`를 추가할 수 있다.
